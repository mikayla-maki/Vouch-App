# Vouch Roadmap

Future enhancements beyond V1. See [VOUCH_ARCHITECTURE.md](./VOUCH_ARCHITECTURE.md) for the current V1 design.

## V2 Enhancements

- Multiple databases per user
- Key rotation events
- Anonymous credentials for multi-writer databases
- Multi-device sync
- Reactions (ThumbsUp, ThumbsDown, Heart, etc.)
- Statistical attack mitigation (improved dummy traffic)
- Safety number verification UI

## V3+ Future

- Tags/categories for recommendations
- Multi-hop trust queries ("friends of friends")
- Block semantics (delete + broadcast disavowals)
- Private information retrieval (multi-server PIR)
- Post-quantum cryptography
- Tor integration
- Hardware security module integration
- Collaborative databases (multiple authorized writers)
- Database forking
